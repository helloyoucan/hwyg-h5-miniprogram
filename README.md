#### 某海鲜交易平台微信小程序版

该项目原为基于vue开发，还实现了一个[React版本](https://github.com/helloyoucan/hwyg-h5-react)。

仅仅简单地实现了几个页面（首页、商品列表、竞拍列表、资讯、个人中心）。
